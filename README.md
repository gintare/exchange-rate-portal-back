# exchange-rate-portal-back
Exchange rate portal backend
Download ZIP and extract to directory
Navigate to pon.xml and open it Intellij IDE
In Intellij navigate to src/main/java com.portal.exchangerate.ExchangerateApplication and run this class
Load backend data by opening http://localhost:3001/data_loader in browser

Now backend is running and quartz technology will take care of to renew data after 24 hours
